# 🔐 Security Policy

## Reporting a Security Issue

OSS Encyclopedia is a documentation-only repository — it contains no executable code, APIs, or services.

However if you find:
- A link pointing to a malicious site
- A repo entry that promotes harmful software
- Any other security concern

Please open an [Issue](../../issues/new) tagged `security` and we will address it promptly.

---

*Thank you for helping keep this project safe and trustworthy! 🛡️*

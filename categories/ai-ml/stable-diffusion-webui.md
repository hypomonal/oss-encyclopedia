## 📌 What it is

The most popular browser-based UI for running Stable Diffusion image generation locally!! 🖼️ Type a text prompt, get a stunning AI-generated image — running 100% on YOUR hardware, completely FREE!! 159,000+ stars and the entire AI art community's go-to tool!!

---

## 🔍 What it does

- 🖼️ Text-to-image generation from any text prompt
- 🔄 Image-to-image transformation and inpainting
- 🎛️ Hundreds of settings, samplers, and fine-tuning options
- 🧩 Massive extension ecosystem — ControlNet, LoRA, upscalers
- 📦 Load any Stable Diffusion model from CivitAI or HuggingFace
- 🌐 Beautiful browser-based UI — no coding needed!!

---

## 🚀 How to use it

```bash
# Clone the repo
git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui
cd stable-diffusion-webui

# Run!! (auto-installs dependencies)
# Windows:
webui-user.bat

# Linux/macOS:
./webui.sh

# Then open http://localhost:7860 in your browser!! 🚀
```

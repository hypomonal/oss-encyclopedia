## 📌 What it is

n8n is a powerful **visual workflow automation** tool — like Zapier but open source, self-hostable, and with AI superpowers!! 🤖 Connect 200+ apps and services together with a beautiful drag-and-drop interface, and add AI steps anywhere in your workflow!!

---

## 🔍 What it does

- 🔗 Connect 200+ apps — Gmail, Slack, GitHub, Notion, databases & more
- 🤖 AI-native — add LLM steps, vector stores, AI agents to any workflow
- 🏠 Self-hostable — your data NEVER leaves your infrastructure
- 💻 Write custom JavaScript/Python nodes when you need full power
- ⏰ Trigger workflows on schedule, webhook, or event
- 🔄 Build complex multi-step automations visually

---

## 🚀 How to use it

```bash
# Run with Docker — easiest way!!
docker run -it --rm \
  --name n8n \
  -p 5678:5678 \
  -v ~/.n8n:/home/node/.n8n \
  docker.n8n.io/n8nio/n8n

# Or with npx
npx n8n

# Open http://localhost:5678 and start building!! 🚀
```

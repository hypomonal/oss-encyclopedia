## 📌 What it is

ComfyUI is a **node-based visual interface** for Stable Diffusion image generation!! 🖼️ Instead of sliders and buttons, you connect nodes like a visual programming language — giving you INSANE control over every step of the image generation pipeline!!

---

## 🔍 What it does

- 🔗 Visual node graph for building complex image generation pipelines
- ⚡ Extremely efficient — faster than AUTOMATIC1111 in many cases
- 🧩 Mix and match models, LoRAs, ControlNets visually
- 💾 Save & share workflows as JSON files
- 🎬 Generate videos with AnimateDiff nodes
- 🖥️ Runs on NVIDIA, AMD, Apple Silicon & even CPU

---

## 🚀 How to use it

```bash
git clone https://github.com/comfyanonymous/ComfyUI
cd ComfyUI
pip install -r requirements.txt

# Run it!!
python main.py

# Open http://localhost:8188 in your browser!! 🚀
```

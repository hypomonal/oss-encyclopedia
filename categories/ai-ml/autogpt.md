## 📌 What it is

AutoGPT was the project that showed the world what **autonomous AI agents** could do!! 🤯 Give it a goal, and it figures out the steps, searches the web, writes code, reads files, and executes tasks — ALL by itself!! The repo that broke the internet in 2023 with 180,000+ stars!!

---

## 🔍 What it does

- 🎯 Give it a high-level goal — it figures out HOW to achieve it
- 🌐 Autonomously browses the web for information
- 💻 Writes and executes code on its own
- 📁 Reads and writes files autonomously
- 🔄 Chains multiple LLM calls together to complete complex tasks
- 🧠 Has long-term memory to track progress across steps

---

## 🚀 How to use it

```bash
git clone https://github.com/Significant-Gravitas/AutoGPT
cd AutoGPT
pip install -r requirements.txt

# Set your API key
cp .env.template .env
# Edit .env with your OpenAI key

# Launch!!
python -m autogpt
```

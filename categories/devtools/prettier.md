## 📌 What it is

Prettier is the **opinionated code formatter** that ended all code style debates FOREVER!! 🕊️ It automatically reformats your code to a consistent style every time you save — no more arguments about tabs vs spaces, semicolons, or line lengths!! Just beautiful, consistent code always!!

---

## 🔍 What it does

- ✨ Auto-formats JS, TS, CSS, HTML, JSON, Markdown & more on save
- 🔥 Eliminates ALL code style debates in your team
- 🔗 Integrates with VS Code, WebStorm, Vim, every editor
- 🤝 Works with ESLint seamlessly
- ⚙️ Minimal config — it's opinionated so you don't have to be!!

---

## 🚀 How to use it

```bash
npm install --save-dev prettier

# Format a file
npx prettier --write src/index.js

# Format everything!!
npx prettier --write "src/**/*.{js,ts,css,html}"

# Check formatting in CI
npx prettier --check "src/**/*.js"

# .prettierrc config (minimal!!)
# { "semi": true, "singleQuote": true, "tabWidth": 2 }
```

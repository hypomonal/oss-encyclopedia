VS Code is the **world's most popular code editor** — used by over 73% of developers globally!! 🌍 Lightweight yet insanely powerful, it works for every language, every stack, every workflow. Microsoft made it open source and free, and the world never looked back!! 176,000+ stars!!

## 🔍 What it does

- ✨ IntelliSense — smart autocomplete for 100+ languages
- 🔍 Powerful search & replace across entire projects
- 🌿 Built-in Git integration — stage, commit, diff without leaving the editor
- 🧩 50,000+ extensions — add ANY functionality you need
- 🐛 Built-in debugger for Node.js, Python, Go & more
- 🌐 Remote development over SSH, Docker, WSL seamlessly
- 🤖 GitHub Copilot integration for AI-powered coding

---

## 🚀 How to use it

```bash
# macOS
brew install --cask visual-studio-code

# Linux (snap)
sudo snap install code --classic

# Windows — download from code.visualstudio.com

# Must-have extensions to install!!
code --install-extension esbenp.prettier-vscode
code --install-extension dbaeumer.vscode-eslint
code --install-extension ms-python.python
code --install-extension GitHub.copilot
```

## 📌 What it is

ESLint is **the JavaScript linter** — it statically analyzes your code to find problems, bugs, and bad patterns BEFORE they ever hit production!! 🛡️ The industry standard for JS/TS code quality, used by virtually every serious JavaScript project on the planet!!

---

## 🔍 What it does

- 🐛 Catches bugs and errors before you run the code
- 📏 Enforces coding standards and best practices
- 🔧 Auto-fixes many issues with `--fix` flag
- 🧩 Hugely extensible — thousands of plugins for React, Vue, TS & more
- 🤝 Works perfectly alongside Prettier
- ⚙️ Fully configurable — enable exactly the rules you want

---

## 🚀 How to use it

```bash
npm install --save-dev eslint

# Initialize config
npx eslint --init

# Lint your files
npx eslint src/

# Auto-fix what it can!!
npx eslint src/ --fix

# eslint.config.js (modern flat config)
export default [
  {
    rules: {
      "no-unused-vars": "error",
      "no-console": "warn",
      "eqeqeq": "error"
    }
  }
];
```

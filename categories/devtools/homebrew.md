## 📌 What it is

Homebrew is **the missing package manager for macOS** (and Linux)!! 🍺 Install any developer tool — Git, Node, Python, PostgreSQL, FFmpeg — with a single `brew install` command!! No more hunting websites and running sketchy installers!!

---

## 🔍 What it does

- 📦 Install 6,000+ packages with one command
- 🖥️ Install GUI macOS apps with `brew install --cask`
- 🔄 Update everything at once with `brew upgrade`
- 🧹 Clean up old versions with `brew cleanup`
- 🍺 Install anything not in the registry with custom taps

---

## 🚀 How to use it

```bash
# Install Homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install anything!!
brew install git node python postgresql redis

# Install GUI apps too!!
brew install --cask vscode docker spotify slack

# Keep everything updated
brew update && brew upgrade

# Search for a package
brew search ffmpeg
```

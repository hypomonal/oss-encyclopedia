## 📌 What it is

Git is the **distributed version control system** that literally runs the entire software world!! 🌍 Created in 2005 by Linus Torvalds in just TWO WEEKS to manage the Linux kernel, it became the universal standard for tracking code changes and collaborating with teams!!

---

## 🔍 What it does

- 📸 Snapshot every change to your codebase forever
- 🌿 Branch and merge without fear — experiment freely!!
- 👥 Collaborate with unlimited developers simultaneously
- ⏪ Roll back any mistake to any point in history
- 🌐 Powers GitHub, GitLab, Bitbucket and every code platform
- 🔍 See exactly who changed what, when, and why

---

## 🚀 How to use it

```bash
# Start a new repo
git init
git add .
git commit -m "🎉 Initial commit!!"

# Collaborate with the world
git clone https://github.com/user/repo
git pull origin main
git push origin main

# Branch safely!!
git checkout -b my-new-feature
git merge my-new-feature

# Time travel — see history!!
git log --oneline --graph
git checkout abc1234  # Go back to any commit!!
```

# 🤝 Contributing to OSS Encyclopedia

First off — **thank you!!** 🎉 Every contribution makes this directory better for thousands of developers worldwide!

---

## ✅ What We Accept

Any repo that is:
- 🟢 Genuinely open source with a valid license (MIT, Apache, GPL, etc.)
- 🟢 Useful to developers, creators, or learners
- 🟢 Actively maintained OR historically significant
- 🟢 Has a public GitHub/GitLab repo
- 🟢 Not already listed!

---

## 🚀 How to Add a Repo

**Step 1** — Fork this repository

**Step 2** — Create a branch
```bash
git checkout -b add/repo-name
```

**Step 3** — Copy the template from [TEMPLATE.md](./TEMPLATE.md) and fill in every field

**Step 4** — Save your file as `categories/CATEGORY/repo-name.md`

**Step 5** — Add the repo to the category's `README.md` index table

**Step 6** — Open a Pull Request with title: `Add: [Repo Name] to [Category]`

---

## 📐 Rules

- ✅ Use the exact template format — no exceptions
- ✅ Code examples must actually work
- ✅ Name files using lowercase with hyphens: `repo-name.md`
- ✅ No self-promotion spam
- ✅ Be honest — don't oversell

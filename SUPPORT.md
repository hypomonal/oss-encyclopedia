# 💬 Support

Need help? Here's how to get it! 😄

---

## 🐛 Found a mistake in an entry?
Open an [Issue](../../issues/new?template=bug-report.md) and tag it `bug`

## ➕ Want to add a repo?
Read [CONTRIBUTING.md](./CONTRIBUTING.md) and open a Pull Request!

## ❓ Have a question?
Open an [Issue](../../issues/new) and tag it `question` — we're friendly!

## 💡 Have a suggestion?
Open an [Issue](../../issues/new) and tag it `enhancement`

---

*We usually respond within a few days. This is a community project maintained with love! 💙*
